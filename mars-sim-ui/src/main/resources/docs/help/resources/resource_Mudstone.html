Resource : "Mudstone" (Solid)


1. Description :

  • A rock succession on the lower slopes of Mount Sharp that comprises thinly laminated mudstones. These rock outcrops are very fine-grained sedimentary rocks that we interpret as having formed from particles suspended in water, which slowly settled.
