Resource : "Allophane" (Solid)


1. Description :

  1. Allophane is an amorphous to poorly crystalline hydrous aluminium silicate clay mineraloid. Its chemical formula is Al2O3·1.3-2·H2O. A regolith derived mineral deposit


2. Manufacturing Processes :

How to make allophane:

  • Process Regolith-b Rich In Poly-hydrated Sulfate
  • Process Regolith-c Rich In Phyllo-silicate
  • Process Regolith-d Basaltic

What to do with allophane:

  • Extract Minerals From Allophane Deposits
