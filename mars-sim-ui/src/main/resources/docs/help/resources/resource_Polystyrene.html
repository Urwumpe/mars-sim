Resource : "Polystyrene" (Solid)


1. Description :

  • When styrene molecules are linked together into a polymer, polystyrene (C8H8)n is created. Polystyrene is an inert plastic that can be used to make many products, such as disposable plates, cups and other foodservice packaging products.
