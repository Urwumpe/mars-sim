Resource : "Polycarbonate resin" (Solid)


1. Description :

  • These are partially bio-based polycarbonates made use of renewable feedstocks (food waste) and CO2. See https://www.bio.org/sites/default/files/Natalie%20Bittner.pdf. They are strong, tough materials, easily worked, molded, and thermoformed.
