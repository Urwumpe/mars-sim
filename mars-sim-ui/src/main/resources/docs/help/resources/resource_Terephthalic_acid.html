Resource : "Terephthalic acid" (Solid)


1. Description :

  • Terephthalic acid is an organic compound with formula C6H4(CO2H)2. This white solid is a commodity chemical, used principally as a precursor to the polyester PET, used to make clothing and plastic bottles.
