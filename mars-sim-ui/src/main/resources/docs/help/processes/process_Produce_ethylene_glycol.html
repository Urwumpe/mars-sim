Process : "Produce ethylene glycol"

1. Description :

  • This is to produce ethylene glycol (CH2OH)2. Ethylene glycol is produced from ethylene (ethene), via the intermediate ethylene oxide. Ethylene oxide reacts with water to produce ethylene glycol according to the chemical equation: C2H4O + H2O → HO−CH2CH2−OH 44 g 18 g 62 g
