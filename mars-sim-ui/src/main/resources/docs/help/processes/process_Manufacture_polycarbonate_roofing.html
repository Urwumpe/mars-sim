Process : "Manufacture polycarbonate roofing"

1. Description :

  • Asahi/Chi Mei produces DPC (polycarbonate) from DMC which is produced via methanolysis of ethylene carbonate. Ethylene carbonate is produced by the reaction between ethylene oxide and carbon dioxide.
