Process : "Produce terephthalic acid"

1. Description :

  • This is to produce terephthalic acid. C8H6O4 Molar mass : 166 In the Amoco process, which is widely adopted worldwide, terephthalic acid is produced by catalytic oxidation of p-Xylene (C8H10): C8H10 + 3O2 --> C8H6O4 + 2H2O 106 g 48 g 166 g 36 g The process uses a cobalt–manganese–bromide catalyst. The bromide source can be sodium bromide, hydrogen bromide or tetrabromoethane.
