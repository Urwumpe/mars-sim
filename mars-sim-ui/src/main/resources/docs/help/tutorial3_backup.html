Tutorial - Step 3 : Menu Bar, Desktop and Status Bar

At the top of the Main Simulation Window is the Menu Bar with headings such as 'File', 'Tools', 'Settings', and 'Help'.

The File menu deals with the functions you are used to in most menu-driven programs: start a New simulation, Save (the default filename is 'default.sim') or Save As (the filename of your choice), and Exit the Simulation

The Tools menu shows a list of eight Tool Windows available for use in mars-sim.

Each Tool Window let you observe and interact the settlement in a particular manner.

You can use an associated function key (F1 to F8) to activate each of them. See User Interface for more details.

The Settings menu lets you choose between full-screen or window mode, skin theme and the volume of background music or sound effects.

The Timestamp Bar displays the current Earth and Martian date and time.

The Calendar button that opens up a Martian calendar.

The Orbit Viewer button opens up a simple solar system view of each planets and buttons for updating the planetary orbits.

The Help menu contains the About, the Tutorial as well as the User Guide.

Directly below the Menu Bar is the Desktop area, where various Tool Windows reside

The Status Bar contains a set of buttons for mars-sim's speed control: a button for increasing speed, a pause button, and a button for decreasing speed, a text label showing mars-sim's memory usage on your computer.
