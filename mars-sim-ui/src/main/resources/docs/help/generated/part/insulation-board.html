Part - Insulation board

Description

Assume each tile is about 1m x 1m. Each board comprises extruded polystyrene (XPS) and expanded polystyrene insulation (EPS) that contain millions of air pockets trapped within the foam. Because the polystyrene itself is also highly resistant to heat, the result is a great thermal insulator which can give excellent R-ratings. XPS is often lined with aluminium foil.
