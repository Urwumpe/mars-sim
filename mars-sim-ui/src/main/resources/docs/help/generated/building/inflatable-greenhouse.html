Building - Inflatable Greenhouse

Description

The Inflatable Greenhouse is essentially a flexible air?bladder made with interwoven layers of Polyurethane, Kevlar, polyester, Saran and Tedlar materials to retain an atmosphere adequate for growing a limited number of crops as the food source for the settlers. It is UV-resistant, bullet-resistant and has an aluminized Mylar reflector curtain. It also houses one workbench with instruments for botany research.
