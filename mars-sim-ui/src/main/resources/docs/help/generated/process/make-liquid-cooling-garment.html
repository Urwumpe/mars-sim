Process - Make Liquid Cooling Garment

Description

The Liquid Cooling and Ventilation Garment (LCVG) serves to regulate the transport water-vapor from the body and maximize thermal conductivity to maintain astronaut comfort. See technical papers at https://www.science.gov/topicpages/v/ventilation+garment+lcvg
