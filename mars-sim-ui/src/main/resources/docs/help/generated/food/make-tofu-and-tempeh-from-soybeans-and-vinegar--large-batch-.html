Food Recipe - Make Tofu and Tempeh from Soybeans and Vinegar (large batch)

Description

Boil Soybeans, blend, separate okara and ferment with Rhizopus oligosporus starter. Add vinegar to soymilk as coagulant. This uses all the okara by-product of tofu.
