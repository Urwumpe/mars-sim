Resource - Aluminized Mylar

Description

Aluminized Mylar is a reflective, electrical insulating material. It contains a coat of aluminum, resulting in a silver-colored radiant barrier that reflects radiant heat and thermal radiation. It shields Space Stations against the thermal radiation of the sun and the bitter cold of space. See 'Efficiency of aluminized mylar insulation at cryogenic temperatures' by James B. Heaney
