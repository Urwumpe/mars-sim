Resource - Prophylene

Description

The common name for the organic chemical compound propene. An alkene which is a colorless gaseous (at room temperature and pressure) hydrocarbon with the chemical formula C3H6. It's a colorless gas with a faint petroleum-like odor and is flammable, and is often used as fuel.
