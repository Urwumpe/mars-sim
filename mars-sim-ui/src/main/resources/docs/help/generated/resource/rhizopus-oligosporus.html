Resource - Rhizopus oligosporus

Description

This edible fungus is used to ferment tempeh. As the fungus grows, its mycelia grow throughout the edible medium, producing an edible block.

Characteristics

Edible:true
Type:Organism
Form:Solid

Used by Food Process

  • Make Tempeh from Soybeans
  • Make Black Oncom from Okara
  • Make Tempeh from half Soybeans and half Okara
  • Make Tofu and Tempeh from Soybeans and Vinegar (large batch)

  • Make Tofu, Soymilk, and Tempeh from Soybeans and Epsom Salt
