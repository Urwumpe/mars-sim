Resource - Gypsum

Description

Gypsum is a soft sulfate mineral composed of calcium sulfate dihydrate, with the chemical formula CaSO4·2H2O. It is widely mined and used as a fertilizer. A typical regolith derived mineral deposit in NASA Water ISRU Study Reference Case B
