Resource - Methanol

Description

Methanol CH3OH can be used as a fuel source, solvent, pesticide, or a chemical feedstock for production of a range of important industrial chemicals such as polyester. It appears as a colorless fairly volatile liquid with a faintly sweet pungent odor like that of ethyl alcohol.
